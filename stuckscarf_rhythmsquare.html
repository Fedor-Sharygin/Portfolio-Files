Rhythm Square

Unity 2021.3.22f

This is a 2D rhythm Unity game, where the main design principle is the simplicity. Game only requires use of one button 'Space' to play through the actual levels. The simplicity also comes through the design of the developer tools (currently in development), where to create a level, all you have to do is to activate the Record Mode (F9), and start pressing the button to the rhythm you yourself set up. After that, simply take the generated level file and replace the previous version with the new content.

In order to motivate people to play as close to the center as they can, I have added a camera based shader to display the amount of points the players earn, using Unity material features, allowing text to be changed during run-time. SFX were also implemented to let the players know whenever they failed to hit the beat, or when they succeeded to do so.

With the code to the right, I have generalized JSON description loading for the 3 main platforms this game was designed for: PC, Android, iOS. This level loading can be used for many different platforms and can load any level set and can be further updated to include platfroms such as PlayStation, XBox, Nintendo Switch and more! This allows development flexibility and simplifies the game development process for a variety of games, as this algorithm does not specify the class that describes the game level, and only asks for a IGameInfo interface, overriden by the coder for whichever implementation is required.
