Game AI - CS580, CS582

CS580 - Game AI classes by Dmitri Volper and Steve Rabin, which teach best approaches to implementing Game AI, including player action reactions, pathfinding (A* + smoothing, NavMeshes), behavior trees

Implemented A* algorithm that utilizes different heuristics (Octile, Chebyshev, Manhattan, Euclidian) to find the most optimal path from the original position to the clicked position.

Collaborated with Matt Haynie on a project that optimized path walking of a game AI character to utilize the most direct path possible using Signed Distance Functions (SDF) for smoothing.

Used knowledge on the subject of Behavior trees on personal projects and Bachelor Project "Valkyrie Quest" (behavior tree below is not mine).
