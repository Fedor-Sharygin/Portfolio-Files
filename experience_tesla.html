Tesla Inc.

UI Software Engineer Intern

08/2021 - 12/2021

I worked with the Tesla tablet UI team developing new features for both users and tablet designers to streamline the process of creating new UI screens, was part of the UI team, where I quickly adapted to the large many year old project and improved my team and programming skills.

I have implemented a new UIML text operator that allows UI designers and interpreters use a simple translation function call inside the UIML element that automatically returns the correct translation, depending on system language parameter. Before this feature, level designers had to be aware of the different tuple and its element names programmer decided to use, if they wanted to return the translation.

This new feature required me to update myself on my knowledge of the Makefile language, while using my understanding of Python to generate the C++ QT file. This file generation happens during compilation with Makefile, which then must also be compiled with the QT compiler, so the interpreter XAML file is generated for translators to fill out.

I have also been tasked with adding C-style math operators to the UIML language. I have used my knowledge of Reverse Polish Notation (RPN) and text interpretation to generate a binary tree of operators to evaluate given the expression. This feature was used for UI designers to place and scale new elements relative to the transform values referred to in the element.
