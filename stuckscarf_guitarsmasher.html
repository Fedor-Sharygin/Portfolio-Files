Guitar Smasher

Unity 2021.3.22f

This is a 3D rhythm Unity game inspired on other rhythm games such as 'Guitar Hero', and 'Dance Dance Revolution'. This project was a proof of concept that I could make and develop a 3D rhythm game, that utilizes my previous works, including 'Rhythm Square', and constructs a game out of each these element. In order to allow players to cater to their control preferences, I have also included the new Input Unity Package that lets me change the user controls at will, while disabling duplicate controls.

As shown on the picture to the left, I have implemented and used Unity Events to create a Timer Component capable of creating looped and delayed effects, being reset / played / stopped through code, similar to the Godot Timer component. This component can be used in many future projects, as the code is independent of other project's scripts.

In addition, I have studied Unity Shader language to create a Unity material that will duplicate the texture depending on how large the scale of the object is. This lets me create and visualize the length of longer Music Ticks without having the textures stretch, which according to playtesters, has worsened the gameplay experience. With this not only did I address the player's complaints to improve the experience, but I have also used my programming skills to learn a new skillset.
