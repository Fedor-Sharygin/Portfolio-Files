Baka Bros Entertainment

Chaos Kart

Chaos Kart - UE5 Kart racing MOBA game with 3 years of development.

I worked closely with Kaelan Simpson, Associate Gameplay Designer, to create and add the General Stat Scaler feature to the game, that allows to scale all stats of cards of the same type/subcategory, to encourage or discourage players to use cards of the given type. I had to implement and track down all of the stat uses both in the client-side (mostly visual) and server-side (real game calculations) within a large number of game modes that utilize different stats, independent of the default values.

Collaborated with the LiveOps team to improve the XAML WWE Mobile game files editor and game behavior. One of the more prevelant bugs was the lack of currency prize randomness, for example, when asked to generate 15 random currencies, 15 currencies of the same type were returned, instead of 15 different types, as expected. Another new feature implemented was the default card parameter editor, where I cooperated closely with the LiveOps team manager to guarantee their expectations were met.

I was tased with implementing multiple tutorials for existing game modes, which included studying, implementing, and following previous and new designs, code standards of various previous developers, and keeping track of client-server communication, as all of the tutorials flags are exclusive to the server.

Closely worked with Aleksey Perfilev on pinpointing and fixing bug in the King of the Ring (KOTR) game mode: when the player has joined the mode's queue, the match was found, and during transition from queue to the game, server broke down, those players would not have their KOTR queue/game flags reset, making it impossible for them to rejoin the queue. We have located the issue and its location in code, however, due to importance of the code placement and unrelated timing issues, we did not have the time to fix it as of my last day.
