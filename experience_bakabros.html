Chaos Kart

04/2023 - 08/2023

Chaos Kart - UE5 Kart racing MOBA game 3 years in development. I joined the team as a volunteer to improve my skills and gain experience with a commerical C++ game engine. I closely collaborated with Daniel Peterson and rest of the crew to create the best multiplayer experience we could, where my main task was to create and improve character abilities, implement UI elements, and create hazards. The gameplay effects were all made using the GAS system to create status states and their effects on the characters and surrounding world, which effect both racing and fighting.

I have made a prototype level called Lava-Castle-Map, where I have implemented my knowledge of level-design techniques to make the level as fun as possible, while showcasing a number of hazards I have implemented consising of fireball and lavapool effects, that can either move to attack the player or are static hazards a player has to avoid, with an addition of arrow volley when approaching the castle section of the map, where the arrow shots are timed to hit the players that enter the AoE, but give them time to avoid the damage.

Implemented a new minimap feature that uses a top-down orthogonal camera that generates a real-time texture projected onto the player's UI, while server calculates the current race positions of each character and passes the info to the users. Each player's logo is placed high above the map, so users can't see it, and the minimap camera can.

Developed character Bucky abilities that have damage, shield, and movement effects. The system utilizes the Unreal Engine GAS system to set the user character's, their allies', and enemies' speed, shield stats to allow boosting/slowdown. The "Hook Throw" ability also allows the player to attack another player and at the same time get a boost to the hooked position or player.
